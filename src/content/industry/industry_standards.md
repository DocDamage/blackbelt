# Industry Standards Reference

## Automotive Industry

### IATF 16949:2016

Automotive Quality Management System

**Core Tools (AIAG):**

| Tool | Purpose | When Used |
|------|---------|-----------|
| APQP | Advanced Product Quality Planning | Product development |
| PPAP | Production Part Approval Process | Before production start |
| FMEA | Failure Mode and Effects Analysis | Design & Process |
| MSA | Measurement System Analysis | Before production |
| SPC | Statistical Process Control | During production |

### PPAP Levels

| Level | Requirements |
|-------|--------------|
| 1 | Part + PSW only |
| 2 | Part + PSW + limited samples |
| 3 | Part + PSW + complete samples + data |
| 4 | PSW + customer requirements |
| 5 | PSW + documents available at site |

**PSW = Part Submission Warrant**

### APQP Phases

1. Plan and Define Program
2. Product Design and Development
3. Process Design and Development
4. Product and Process Validation
5. Feedback, Assessment, Corrective Action

### Automotive Cpk Requirements

| Characteristic | Minimum Cpk |
|----------------|-------------|
| Safety-critical | ≥ 1.67 |
| Key characteristic | ≥ 1.33 |
| Standard | ≥ 1.00 |

## Healthcare / Medical Devices

### ISO 13485:2016

Medical Devices Quality Management System

**Key Requirements:**

- Design and development controls
- Risk management (ISO 14971)
- Traceability of products
- Validation of processes
- Post-market surveillance

| Subpart | Topic |
|---------|-------|
| B | Quality System Requirements |
| C | Design Controls |

| L | Handling, Storage, Distribution |
| M | Records |

### Design Controls (820.30)

```
User Needs → Design Input → Design Process → Design Output → Medical Device
                                    ↓
                             Design Review
                             Verification (Did I build it right?)
                             Validation (Did I build the right thing?)
                             Transfer
```

### CAPA Requirements

- Analyzing processes and quality data
- Identifying product and quality problems
- Investigating root causes
- Identifying corrective action
- Verifying/validating actions
- Implementing and recording changes
- Disseminating information
- Submitting relevant information for management review

## Pharmaceutical / Life Sciences

### GMP (Good Manufacturing Practice)

**21 CFR Parts:**

| Part | Coverage |
|------|----------|
| 210 | Current GMP in manufacturing |
| 211 | Current GMP for finished pharma |
| 212 | Positron emission tomography drugs |
| 600-680 | Biologics |

### Process Validation (FDA Guidance)

**Three Stages:**

1. **Process Design** - Build quality into process
2. **Process Qualification** - Demonstrate reproducibility
3. **Continued Process Verification** - Ongoing assurance

### 21 CFR Part 11 (Electronic Records)

**Requirements for:**

- Electronic signatures
- Audit trails
- System validation
- Security controls
- Time-stamped records

### Data Integrity (ALCOA+)

| Letter | Meaning |
|--------|---------|
| A | Attributable (who did it) |
| L | Legible (can be read) |
| C | Contemporaneous (recorded when done) |
| O | Original (or true copy) |
| A | Accurate (correct and complete) |
| + | Complete, Consistent, Enduring, Available |

## Aerospace

### AS9100D

Aerospace Quality Management System

**Special Requirements:**

- Configuration management
- Risk management
- Counterfeit parts prevention
- First Article Inspection (FAI)
- Special process control
- Product safety

### NADCAP (National Aerospace and Defense Contractors Accreditation)

**Special Processes:**

- Heat treating
- Welding
- Non-destructive testing (NDT)
- Chemical processing
- Coatings
- Composites
- Elastomer seals

### First Article Inspection (AS9102)

Three forms:

1. **Form 1**: Part Number Accountability
2. **Form 2**: Product Accountability (material, processes)
3. **Form 3**: Characteristic Accountability (dimensions, features)

**Program Elements:**

### RoHS / REACH / WEEE

| Regulation | Focus |
|------------|-------|
| RoHS | Restricts hazardous substances in electronics |
| REACH | Chemical registration and authorization |
| WEEE | Waste electrical and electronic equipment |

**RoHS Restricted Substances:**

- Lead (Pb) < 0.1%
- Mercury (Hg) < 0.1%
- Cadmium (Cd) < 0.01%
- Hex Chrome (Cr6+) < 0.1%
- PBB < 0.1%
- PBDE < 0.1%
- DEHP, BBP, DBP, DIBP < 0.1% (added 2019)

## Food & Beverage

### HACCP (Hazard Analysis Critical Control Points)

**7 Principles:**

1. Conduct hazard analysis
2. Determine Critical Control Points (CCPs)
3. Establish critical limits
4. Establish monitoring procedures
5. Establish corrective actions
6. Establish verification procedures
7. Establish documentation

### FSSC 22000

Food Safety System Certification

- ISO 22000 base
- Sector-specific PRPs
- Additional FSSC requirements

### SQF (Safe Quality Food)

**Certification Levels:**

- Level 1: Food Safety Fundamentals
- Level 2: HACCP-based Food Safety Plans
- Level 3: Food Safety and Quality Management

### Risk-Based Thinking

All modern standards emphasize:

- Identify risks and opportunities
- Plan actions to address risks
- Implement actions
- Evaluate effectiveness
